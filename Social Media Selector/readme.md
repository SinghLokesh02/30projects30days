# Social Media Selector

The **Social Media Selector** project is a simple and intuitive dropdown menu that allows users to select their preferred social media platform. The UI includes popular platforms like Facebook, Instagram, Twitter, YouTube, and WhatsApp, each represented by recognizable icons.

The **Social Media Selector** is a small yet stylish component designed for web pages where users may want to quickly access or select a social media platform. It's built with HTML, CSS, and JavaScript, making it fully customizable and easy to integrate into any project.

- Dropdown menu with popular social media icons
- Smooth transition when toggling the dropdown
